# 🏰 Realms of Myth

A fantasy Minecraft add-on featuring playable races, class-based abilities, dragon boss battles, and tiered fantasy weapons & armor.

**Compatible with:** Minecraft Education Edition & Bedrock Edition

---

## Features (Planned)
- 🧝 4 Playable Races (Elf, Troll, Giant, Human)
- ⚔️ 5 Classes (Mage, Ranger, Berserker, Paladin, Druid)
- 🐉 Dragon Boss Fights (Fire & Frost)
- 🛡️ Tiered Weapons & Armor (Iron → Mythril → Dragon Bone → Legendary)
- 📜 Class Selection System at game start

## Status
🚧 Early development — scaffolding phase
